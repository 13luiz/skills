# Agent Skills

A collection of agent skills that extend capabilities across AI engineering, development, and tooling.

## AI Engineering

These skills help you build and maintain high-quality AI-assisted development workflows.

- **harness-engineering-guide** — Audit, design, and implement AI agent harnesses for any codebase. Three modes: Audit (scorecard across 8 dimensions), Implement (set up AGENTS.md, CI, lint rules), Design (full harness strategy). Supports 17 project types, 11 language ecosystems, and 3 lifecycle stages.

  ```
  npx skills@latest add 13luiz/skills/harness-engineering-guide
  ```
